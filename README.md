# SimpleMusic

> 网易云音乐高仿(部分实现)\
> 安卓大作业\
> 顺便接触强类型语言, 接触类的使用, 以便后期转入TS\
> 接触安卓组件开发方式及其行为, 还有其资源管理

### 实现功能

0.歌单创建/删除/收藏\
1.歌单添加/删除歌曲\
2.歌曲播放\
3.推荐列表
